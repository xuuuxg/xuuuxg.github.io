记复习Java

数组

一维数组

int[] arr = {0,1,2,3}
for(int i = 0; i< arr.length; i++) {
	arr[i];
}

二维数组

int[][] arr = new int[10][10];

在创建数组的过程中会做这么几件事情：

  1. 创建一个长度为10的数组

    int[][] arr = new int[10][]

  2. 遍历再创建长度为10的数组

  3. 将2创建的数组赋值为0

    for(int i = 0; i< 10; i++) {
	arr[i] = new arr[10];
}

我理解二维数组其实是一个变相的一维数组。

方法变量的修饰符

  1. public

    修饰方法、变量,所有类、方法都可以访问

  2. protected

    修饰方法、变量,同包下、当前类、父子类可以访问

  3. 默认的

    修饰方法、变量,同包下、当前类可以访问

  4. private

    修饰方法、变量,当前类可以访问

this

  1. Java中的this都是隐式调用，如果明确调用自己类中的属性可以用到this。

    public class PerSon {

       private int id;

       private int getId(int id) {
           return this.id = id;     
           //这里的this就是描述调用类本身的id
       }

       //忽略set
   }

  2. 构造器内调用另一个构造器，会用到this。

    @Data
public class PerSon {

	private int id;
	private String name;
	
	public PerSon() {
		this.Cirl(id);
	}
	
	public PerSon(int id) {
		this.id = id;
	} 
	
	public PerSon(int id, String name) {
		this.id = id;
		this.name = name;
	}
}

    this的限制：只能在构造器中第一行出现

  3. 可以在代码块中使用

    @Data
public class PerSon {
	{
		this.setId(1);
	}
	{
		this.setName("小明");
	}
	
	private int id;
	
	private Stirng name;
}

    上述例子不具备实际开发中的参考，只是示例

扩展类

  1. 子类继承超类

    使用super关键字调用超类构造器

    //超类
@Data
public class PerSon {

	private int id;
	
	private String name;
	
	public PerSon(){}
	
	public PerSon(String name) {
		this.name = name;
	}
}
    //子类
public class Man {

	private int sex;
	
	public Man() {}
	
	public Man(String name, int sex) {
		super(name);        // 使用超类name属性
		this.sex = sex;
	}
}

  2. 多超类的this与super

    // 父类
@Data
public class A {
	private int id;
}

// B继承A
@Data
public class B extends A{
	private int id;
}

// C继承B
@Data
public class C extends B{
	private int id;
}

    在C内调用方式：

    id C的id

    this.id C的id

    super(id) B的id

    ((B)this).getId() B的id

    ((A)this).getId() A的id
