RabbitMQ学习

安装

  1. 安装Erlang依赖

Erlang下载官网 下载对应系统的安装包

安装完毕后安装RabbitMQ Server

RabbitMQ官网

如果不安装RabbitMQ Server就会出现连接错误

启动可视化

  1. 进入安装目录

    • 我安装的是D盘 进入 D:\RabbitMQServer\rabbitmq_server-3.8.4\sbin

    • 查看用户

      rabbitmqctl list_users

      默认有一个guest用户，默认密码guest。这个用户有mq所有的权限。

      我们也可以创建一个自己的用户

      rabbitmqctl add_user  xxg  xxg       --创建用户 xxg  密码为 xxg 

rabbitmqctl  set_permissions  xxg ".*"  ".*"  ".*"    --赋予xxg读写所有消息队列的权限

rabbitmqctl  set_user_tags xxg administrator    --分配用户组

      开启插件：

      rabbitmq-plugins enable rabbitmq_management

      查看地址 http://IP:15672 使用创建的用户名进行登录

实现

常用交换机模式，分别为 默认交换机、直连交换机、扇形交换机、主题交换机。

当消费者掉线，消息也不会随之丢失。

  • 首先Maven导入依赖
<dependency>
	<groupId>org.springframework.boot</groupId>
	<artifactId>spring-boot-starter-amqp</artifactId>
</dependency>
  • 配置Config
import org.springframework.amqp.rabbit.config.SimpleRabbitListenerContainerFactory;
import org.springframework.amqp.rabbit.connection.CachingConnectionFactory;
import org.springframework.amqp.rabbit.connection.ConnectionFactory;
import org.springframework.amqp.rabbit.core.RabbitTemplate;
import org.springframework.amqp.support.converter.Jackson2JsonMessageConverter;
import org.springframework.context.annotation.Bean;

public class RabbitMQConfig {

    @Bean
    public ConnectionFactory connectionFactory() {
        CachingConnectionFactory connectionFactory = new CachingConnectionFactory();
        connectionFactory.setAddresses("localhost:5672");
        connectionFactory.setUsername("xxg");
        connectionFactory.setPassword("xxg");
        connectionFactory.setPublisherConfirms(true);//消息确认
        connectionFactory.setPublisherReturns(true);
        return connectionFactory;
    }

    @Bean
    public RabbitTemplate rabbitTemplate(ConnectionFactory connectionFactory) {
        RabbitTemplate template = new RabbitTemplate(connectionFactory);
        template.setMessageConverter(new Jackson2JsonMessageConverter());
        return template;
    }

    @Bean
    public SimpleRabbitListenerContainerFactory rabbitListenerContainerFactory(ConnectionFactory connectionFactory) {
        SimpleRabbitListenerContainerFactory factory = new SimpleRabbitListenerContainerFactory();
        factory.setConnectionFactory(connectionFactory);
        factory.setMessageConverter(new Jackson2JsonMessageConverter());
        return factory;
    }
}
  1. 默认交换机

默认交换机不需要routingKey来绑定路由，队列名称就是routingkey

生产者

@Component
public class Send {

    private final static String QUEUE_NAME = "hello";

    @Autowired
    private RabbitTemplate rabbitTemplate;


    public void send(String msg) {

        rabbitTemplate.convertAndSend(this.QUEUE_NAME, msg);
    }
}

消费者

@Component
@RabbitListener(
        queuesToDeclare = @Queue("hello")
        )
public class RabbitOnMessage {

    @RabbitHandler
    public void onMessage(@Payload String msg) {
        System.out.println(msg);
    }

}

每当消息生产时后，消费者监听到后就会将该消息消费掉

  1. 直连交换机

生产者

@Component
public class Send {

    private final static String EXCHANGE = "hello";

    @Autowired
    private RabbitTemplate rabbitTemplate;


    public void send(String msg) {

        rabbitTemplate.convertAndSend(this.EXCHANGE,"direct", msg);
    }
}

消费者

@Component
@RabbitListener(
        bindings = @QueueBinding(
        value = @Queue("DirectQueue"),
        exchange = @Exchange(value = "hello", type = ExchangeTypes.DIRECT),
        key = "direct"
))
public class RabbitOnMessage {

    @RabbitHandler
    public void onMessage(@Payload String msg) {
        System.out.println(msg);
    }
}

每当消息生产时后，消费者监听到后就会将该消息消费掉

  1. 扇形交换机

广播模式

通过交换机名称当做routingKey进行队列绑定

消费者

@Component
@RabbitListeners({
        @RabbitListener(
                bindings = @QueueBinding(
                        value = @Queue("fountQueue1"),
                        exchange = @Exchange(value = "fountHello", type = ExchangeTypes.FANOUT),
                        key = "one")),
        @RabbitListener(
                bindings = @QueueBinding(
                        value = @Queue("fountQueue2"),
                        exchange = @Exchange(value = "fountHello", type = ExchangeTypes.FANOUT),
                        key = "two") )
})
public class RabbitOnMessage {

    @RabbitHandler
    public void onMessage(@Payload String msg) {
        System.out.println(msg);
    }
}

生产者

@Component
public class Send {

    private final static String EXCHANGE = "fountHello";

    @Autowired
    private RabbitTemplate rabbitTemplate;


    public void send(String msg) {
		// 第一个参数：交换机，第二个参数：routingKey（随意），第三个参数为消费字段
        rabbitTemplate.convertAndSend(this.EXCHANGE, "info",msg);
    }
}

每当消息生产时后，消费者监听到后就会将该消息消费掉

  1. 主题交换机

Topics 模式

创建多个消费者，通过routingKey条件匹配进行消费

为匹配一个单词（经实际操作可以匹配两个单词）比如exchange 为 hello.test.topic，匹配规则为 *.topic 也可以匹配

#为匹配一个或多个单词

生产者

@Component
public class Send {

    private final static String Exchange = "topic";
    private final static String ROUTINGKEY1 = "hellop.hello.test.topic";
    private final static String ROUTINGKEY2 = "hellop.hello.test1.topic";

    @Autowired
    private RabbitTemplate rabbitTemplate;


    public void send(String msg) {

        rabbitTemplate.convertAndSend(this.Exchange, this.ROUTINGKEY1,msg);
    }

    public void send2(String msg) {

        rabbitTemplate.convertAndSend(this.Exchange, this.ROUTINGKEY2,msg);
    }
}

消费者

@Component
public class RabbitOnMessage {
    // 匹配一个单词
    @RabbitListener(
         bindings = @QueueBinding(
                 value = @Queue("onMessage1"),
                 exchange = @Exchange(value = "topic", type = ExchangeTypes.TOPIC),
                 key = "hellop.*.test1.*"
         )
    )
    @RabbitHandler
    public void onMessage1(@Payload String msg, @Headers Map<String, Object> headers) {
        System.out.println(headers.get(AmqpHeaders.CONSUMER_QUEUE)+"---"+msg);
    }

    // 匹配多个单词
    @RabbitListener(
            bindings = @QueueBinding(
                    value = @Queue("onMessage2"),
                    exchange = @Exchange(value = "topic", type = ExchangeTypes.TOPIC),
                    key = "#.topic"
            )
    )
    @RabbitHandler
    public void onMessage2(@Payload String msg, @Headers Map<String, Object> headers) {
        System.out.println(headers.get(AmqpHeaders.CONSUMER_QUEUE)+"---"+msg);
    }

}
